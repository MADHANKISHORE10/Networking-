DHCP messages were captured using Wireshark during IP address allocation. The process begins with a DHCP Discover message sent by the client to locate a DHCP server. The server responds with a DHCP Offer containing an available IP address. The client then sends a DHCP Request to accept the offered IP, and finally, the server sends a DHCP Acknowledge message to confirm the assignment. This sequence demonstrates the DORA process used for dynamic IP configuration.
The DHCP process was captured using Wireshark by releasing and renewing the IP address. Initially, a DHCP Request and Acknowledge were observed during the renew process. After releasing the IP address, a full DHCP cycle was triggered, including Discover, Offer, Request, and Acknowledge messages. This demonstrates how a client obtains an IP address dynamically using the DHCP protocol.
